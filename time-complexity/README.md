# Time Complexsity
## asymptotic notations:
- Omega notation -> Best case
- theta notation -> Average case
- Big O notation -> Worst case

## rules to calculate time complexsity
- Always take that woest case
- ignore the constant 

### constant: O(1)
### logarithomic complexsity: O(logN)
### sqrt complexity: O(sqrt(n))
### linear complexity: O(n)
### linearthomic: O(NlogN)
### quadreatic complexity: O(n*n)
